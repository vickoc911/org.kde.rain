# org.kde.rain
Live wallpaper that simulates falling rain

How to install it:

- Download it from GitHub.

- Unzip it and rename the folder to org.kde.rain

- Copy the folder to .local/share/plasma/wallpapers/ in your home directory.

- That's it.
